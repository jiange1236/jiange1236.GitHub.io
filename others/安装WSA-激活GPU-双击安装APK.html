                                                                                                    # 安装WSA-激活GPU-双击安装APK

                                                                                                    仅支持Windows11（包含正式版、RP、Beta、Dev） 》》》 目 录 《《《 1.安装WSA本体（必装） 2.激活GPU（普通用户请跳过！！！） 3.安装“安装程序”（选装） 4.安装ADB（选装） 5.安装Android应用 6.卸载Android应用 7.注意事项 8.参考链接

                                                                                                    【1 安装WSA本体】 - ① 微软商店程序抓包网址： 查看链接open in new window ② WSA微软商店链接： 查看链接open in new window - 1.打开①链接，在输入框中输入②，选择Slow通道，确认，下载最底下的"MicrosoftCorporationII.WindowsSubsystemForAndroid_1.7.32815.0_neutral_~_8wekyb3d8bbwe.msixbundle" (注：四个选项与Windows渠道相对应，Fast-Dev渠道，Slow-Beta渠道，RP-RP渠道和Retail-正式版，目前只有Beta版本有发布)

                                                                                                    输入下载地址，选择正确通道

                                                                                                    选择最新安装包

                                                                                                    2.右键Windodws徽标，选择"Windows终端（管理员）"输入命令 add-appxpackage "文件路径/文件名"，回车，安装完成

                                                                                                    打开终端

                                                                                                    输入指令

                                                                                                    3.开启主板虚拟化支持，具体按照各BIOS进行设置（新购买电脑一般已开启主板虚拟化支持） 4.系统中开启Hyper-V和虚拟机平台，打开资源管理器，在路径中输入中文"控制面板"，回车，即可进入控制面板，选择程序-程序和功能-启用或关闭Windows功能，开启"Hyper-V"和"虚拟机平台"，确定，重启

                                                                                                    资源管理器可以直接进入控制面板

                                                                                                    按图开启Windows功能

                                                                                                    5.完成，Enjoy it.

                                                                                                    【2 激活GPU】（普通用户无需进行该项设置！！！） - INTEL英特尔： 查看链接open in new window NVIDIA英伟达（经验证无效，请使用官网最新驱动）： 查看链接open in new window AMD超微半导体： 查看链接open in new window opengl3支持： 查看链接open in new window - 1.下载安装对应显卡驱动和opengl3支持，安装后重启 （注：具体是否成功受概率等玄学因素影响，Intel驱动如果安装失败可尝试再次安装；未测试是否影响调用，建议普通用户跳过该步骤） 2.设置中开启高性能模式，设置-系统-显示-图形，应用的自定义选项-浏览，添加WSA，选择WSA，图形首选项选择高性能，保存

                                                                                                    打开设置

                                                                                                    添加WSA

                                                                                                    选项中选择高性能模式

                                                                                                    3.显卡设置中开启高性能模式（以N卡为例，集显和A卡请自测），NVIDIA控制面板-管理3D设置-添加-选择WSA，确定，并调整WSA首选图形处理器为高性能NVIDIA处理器

                                                                                                    打开NVIDIA控制面板并添加WSA

                                                                                                    选择高性能NVIDIA处理器

                                                                                                    4.重启，完成

                                                                                                    安兔兔GPU显示分数，但很低

                                                                                                    【3 安装“安装程序”】 - APKInstaller： 查看链接open in new window

                                                                                                    安装程序可以直接桌面显示apk且可以双击安装

                                                                                                    以WinUi的界面进行安卓程序的安装

                                                                                                    - 1.下载最新压缩包，如"APKInstaller.Package._0.0.1.0_Test.rar" 2.解压缩 3.开启旁加载模式，设置-隐私和安全性-开发者选项-开启"从任意源（包括松散文件）安装应用"

                                                                                                    打开开发人员模式

                                                                                                    4.右键msixbundle文件，如"APKInstaller (Package)_0.0.1.0_x86_x64_arm64.msixbundle"，属性-数字签名-双击签名列表中的签名-查看证书-安装证书-本地计算机-将所有的证书都放入下列储存-受信任人-完成

                                                                                                    打开属性

                                                                                                    安装程序证书 1

                                                                                                    安装程序证书 2

                                                                                                    5.双击msixbundle文件，如"APKInstaller (Package)_0.0.1.0_x86_x64_arm64.msixbundle"，自动安装框架，然后提示安装失败 6.右键"Install.ps1"选择"使用PowerShell运行"

                                                                                                    安装程序本体

                                                                                                    7.安装完成

                                                                                                    【4 安装ADB】 - 最新ADB： 查看链接open in new window - 1.下载解压缩，将文件夹内的所有文件复制到"C:\Users\用户名"下即可

                                                                                                    安装adb在正确的位置，无需配置环境变量

                                                                                                    【5 安装Android应用】 - 方法一（ADB安装）： 1.打开"Windows Subsystem for Android™"，打开"开发人员模式"

                                                                                                    打开WSA

                                                                                                    开启开发人员模式，注意未启动WSA不能安装程序

                                                                                                    启动WSA

                                                                                                    启动成功，可以安装应用

                                                                                                    2.确保已正确安装ADB，Win+R，输入"CMD"，回车，输入指令"adb connect 127.0.0.1:58526" 3."adb install （将安装文件拖入此处）"，回车，安装完成 - 方法二（安装器安装）： 1.确保已正确安装APKInstaller，双击.apk的安卓程序进行安装，根据指引打开"开发人员模式" （注：Android子系统必须处于开机状态才能进行安装，可以点击WSA中"开发人员模式"下"管理开发人员设置"唤醒）

                                                                                                    【6 卸载Android应用】 - 1.列表中直接右键卸载即可
