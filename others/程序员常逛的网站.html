# 程序员常逛的网站

# 一、在线教程

首先列出一些在线教程网站，这些在线教程网站通常都比较适合入门，可以作为开发学习路上的第一个阶梯，也可以作为工作中的在线文档。

# 1、http://how2j.cnopen in new window

  • 地 址：https://how2j.cn/open in new window

  • 简 介：一个 Java 全栈开发教程网站，内容全面，简洁易懂，非常适合入门。

  • 推荐指数：⭐⭐⭐⭐⭐

# 2、w3cschool

  • 地 址：https://www.w3cschool.cn/open in new window

  • 简 介：前端和脚本语言为主的在线教程网站，前端的内容非常不错。

  • 推荐指数：⭐⭐⭐⭐⭐

# 3、菜鸟教程

  • 地 址：https://www.runoob.com/open in new window

  • 简 介：和 w3cschool 类似的在线教程网站，前端学习看这两个网站就够了。

  • 推荐指数：⭐⭐⭐⭐⭐

# 4、易百教程

  • 地 址：https://www.yiibai.com/open in new window

  • 简 介：内容比较全面的在线教程网站。

  • 推荐指数：⭐⭐⭐

# 9、并发编程网

  • 地 址：http://ifeve.com/open in new window

  • 简 介：比较不错的技术网站，以Java为主，关注并发、NIO、JVM、框架等方面的内容。

  • 推荐指数：⭐⭐

# 10、http://jenkov.comopen in new window

  • 地 址：http://tutorials.jenkov.com/open in new window

  • 简 介：一个国外的技术网站，提供了非常不错的Java教程。

  • 推荐指数：⭐⭐⭐

# 11、http://baeldung.comopen in new window

  • 地 址：https://www.baeldung.com/open in new window

  • 简 介：也是一个国外的技术网站，提供了非常不错的Java教程。

  • 推荐指数：⭐⭐⭐

# 二、视频教程

上面的在线教程网站以文字性内容为主，同样收藏了一些以视频为主的一些网站，相比较而言，更容易上手，当然，也更加耗时间。

# 1、B站

  • 地 址：https://www.bilibili.com/open in new window

  • 简 介：没错，bilibili也可以用来学习，除了番剧、鬼畜之外，b站同样有非常多的高质量的编程教学视频，搜索就行了。

  • 推荐指数：⭐⭐⭐⭐⭐

# 2、慕课网

  • 地 址：https://www.imooc.com/open in new window

  • 简 介：比较不错的编程视频教学网站，可以找到比较体系的东西，当然，有些内容要花钱就是了。

  • 推荐指数：⭐⭐⭐⭐

# 3、中国大学MOOC

  • 地 址：https://www.icourse163.org/open in new window

  • 简 介：有很多名校的课程，当然不止CS。

  • 推荐指数：⭐⭐⭐⭐

# 4、网易云课堂

  • 地 址：https://study.163.comopen in new window

  • 简 介：有一些计算机编程方面的视频教程。

  • 推荐指数：⭐⭐⭐

# 5、实验楼

  • 地 址：https://www.lanqiao.cn/courses/open in new window

  • 简 介：IT技术课程网站。

  • 推荐指数：⭐⭐

# 6、我要自学网

  • 地 址：（敏感词，自行百度）

  • 简 介：比较多的自学教程，有程序设计内容的分类。

  • 推荐指数：⭐⭐

# 7、大学生自学网

  • 地 址：http://v.dxsbb.com/jisuanji/open in new window

  • 简 介：有一些计算机软件的大学课程。

  • 推荐指数：⭐⭐

# 8、极客学院

  • 地 址：https://www.jikexueyuan.com/open in new window

  • 简 介：IT编程课程网站，付费内容居多。

# 四、官网

毫无疑问，一项技术，最权威的文档一定是它的官方文档，所有的教程、书籍都是直接或者间接在官方相关文档的基础上完成。所以，编程学习，后面最好还是要看官网的。当然，官网大部分都是英文——其实配合一些翻译插件，没有想象中的那么难。

# 1、Java

  • 地 址：https://www.java.com/zh_CN/open in new window 、https://docs.oracle.com/en/java/javase/index.htmlopen in new window

  • 简 介：这个不需要多介绍，Java官网和Java官方API。

  • 推荐指数：⭐⭐⭐⭐⭐

# 2、Spring

  • 地 址：https://spring.io/open in new window

  • 简 介：毫无疑问Spring已经成为Java EE事实上的标准。

  • 推荐指数：⭐⭐⭐⭐⭐

# 3、MySQL

  • 地 址：https://www.mysql.com/open in new window

  • 简 介：MySQL官网

  • 推荐指数：⭐⭐⭐⭐

# 4、MyBatis

  • 地 址：https://mybatis.org/mybatis-3/zh/index.htmlopen in new window

  • 简 介：国内最流行的Java持久层框架，而且幸运的是，MyBatis文档是有中文版的。

  • 推荐指数：⭐⭐⭐⭐

# 5、Vue.js

  • 地 址：https://cn.vuejs.org/open in new window

  • 简 介：国内最流行的前端SOP框架，Vue.js的文档是中文的。

  • 推荐指数：⭐⭐⭐⭐

# 6、Linux

  • 地 址：https://www.linux.org/open in new window

  • 简 介：后端程序员必会的Linux.

  • 推荐指数：⭐⭐⭐⭐

# 7、Git

  • 地 址：https://git-scm.com/open in new window

  • 简 介：版本管理推荐用Git，同样有中文版本的文档。

  • 推荐指数：⭐⭐⭐⭐

# 8、Dubbo

  • 地 址：http://dubbo.apache.org/zh-cn/open in new window

  • 简 介：一款高性能的Java RPC框架，国内用的还是比较广泛，源于阿里，中文文档是有的。

  • 推荐指数：⭐⭐⭐⭐

# 9、Redis

  • 地 址：https://redis.io/open in new window

  • 简 介：最流行的No SQL数据库。

  • 推荐指数：⭐⭐⭐⭐

# 五、国内博客社区
